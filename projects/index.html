Projects

These are some of the projects I have worked on in my spare time. Each project started with me either wanting to scratch an itch, or try out a specific technology.

Backbone Index

I built Backbone Index because I wanted to learn Backbone.js, and wanted an easy way to discover plugins and resources. It works by transforming the Backbone wiki into JSON, which is then loaded into the Backbone application. Try it for yourself or have look at the source code.

Technologies: Backbone, jQuery, Ruby, CoffeeScript and Sass.

Measurehub

Measurehub is a web application for tracking any sort of metric. Think of it as a simplified spreadsheet and graph solution. It enables sharing experiments with friends to keep up motivation and support. Try it today!

Technologies: Ruby on Rails, JRuby, Slim, Sass, CoffeeScript, D3.js and Heroku.

Chromarks

Chromarks is a RubyGem for searching chrome bookmarks from the terminal. Check out the source code on Github.

Technologies: Ruby and RSpec

HackerNews Sorter Chrome Extension

I like reading technology news on HackerNews, and to be able to focus on trending stories, I wrote a Chrome extension to sort the posts by who many points they have. Check out the source code on Github.

Technologies: Chrome Extensions API, JavaScript and CSS.

Stay on Track

In 2011 I wrote a small Rails application to keep track on when I went to the gym for an entire year. The premise is that creating a chain of events helps build a habit. It worked out great, but I do not maintain or use it anymore though.

Technologies: Ruby on Rails, PostgreSQL, Haml, Sass, Devise, Heroku and RSpec.
